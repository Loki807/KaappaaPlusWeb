Privacy Policy & Terms of Service

1. Introduction

Kaappaan is a safety assistance application developed to help users access support services. This Privacy Policy explains how we collect, use, and protect your information.

2. Information We Collect

  • Location (only when the user triggers an SOS request)
  • Device information
  • Account details (name, phone number, email)
  • Basic usage logs

3. How We Use Information

  • To improve app performance
  • To provide location-based support if user requests help
  • To maintain account security

4. Security Measures

  • Encrypted communication
  • Secure storage practices
  • Strict access control

5. User Rights

  • Right to request account deletion
  • Right to update information
  • Right to decline optional data access
